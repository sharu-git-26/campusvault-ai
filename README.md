# CampusVault AI (PG Level MCA Project)

An AI-powered academic resource sharing platform with:
- JWT Authentication & Profiles
- College-based Private/Public Access Control
- AI Semantic Search (TF-IDF)
- Upload, Review & Rating System
- Filters & Tags
- Admin Analytics Dashboard

Tech Stack:
- FastAPI (Backend)
- React + Vite (Frontend)
- SQLite (Database)
- Docker (Deployment)

Run Locally:
Backend:
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

Frontend:
cd frontend
npm install
npm run dev
